### Задание 3
```mermaid
---
title: ЛР2(№3) Бурцев 323СА1
---
flowchart TD
    Start(["Старт"]) --> VvodLogin[/"Введите логин и IP-адрес сервера"/]
    VvodLogin --> InitConnect[Инициализация TCP-соединения]
    InitConnect --> Obmen[Обмен версиями протокола и\nпараметрами шифрования]
    Obmen --> AuthMethod{Сервер поддерживает аутентификацию по паролю?}
    AuthMethod -- Нет --> NotSupported[Метод не поддерживается]
    NotSupported --> VivodError1[/"Ошибка аутентификации"/]
    VivodError1 --> End(Конец)
    AuthMethod -- Да --> Setup["Попытки = 3"]
    Setup --> CheckAttempts{"Попытки > 0"}
    CheckAttempts -- Да --> VvodPass[/"Введите пароль"/]
    VvodPass --> SendPass[Отправка пароля серверу]
    SendPass --> CheckPass{Пароль верный?}
    CheckPass -- Да --> AuthSuccess[Аутентификация успешна]
    AuthSuccess --> StartSession[Начало SSH-сессии]
    StartSession --> VivodSuccess[/"Подключение установлено"/]
    VivodSuccess --> End
    CheckPass -- Нет --> Unluck["Попытки - 1"]
    Unluck --> CheckAttemptsLeft{"Попытки > 0?"}
    CheckAttemptsLeft -- Нет --> NoAttempts[Попытки исчерпаны]
    NoAttempts --> VivodError2[/"Доступ запрещен"/]
    VivodError2 --> End
    CheckAttemptsLeft -- Да --> CheckTimeout{Сервер назначил таймаут?}
    CheckTimeout -- Нет --> CheckAttempts
    CheckTimeout -- Да --> SetTimeout[Таймаут]
    SetTimeout --> WaitTimeout[Ожидание истечения таймаута]
    WaitTimeout --> Setup
```
